Predicting Movie Success with Machine Learning (Supervised and Unsupervised Models)

This project explores the use of both supervised and unsupervised learning techniques to analyze and predict the financial success of movies using real-world movie metadata. The dataset includes key features such as budget, revenue, popularity, vote count, runtime, and average rating. The primary objective was to build a predictive model that determines whether a movie is likely to be financially successful (i.e., revenue exceeds budget), while also using unsupervised learning to identify patterns and groupings within the movie data.

Supervised Learning: Classification Models

To classify movies as either "successful" or "not successful," I implemented and compared several supervised learning algorithms, including Logistic Regression, Naive Bayes, and Decision Tree Classifier. Each model was trained using feature sets tailored to its underlying assumptions. Logistic Regression and Naive Bayes were applied to scaled or binarized numerical features, while the Decision Tree model leveraged raw, unscaled data to preserve interpretability. During development, separate train-test splits were used for each modelâ€™s unique preprocessing pipeline. For final evaluation, all models were retrained using a consistent 70/30 disjoint split. Logistic Regression achieved 78% accuracy, followed by Decision Tree at 72% and Naive Bayes at 71%.

Unsupervised Learning: Clustering and Pattern Discovery

To complement classification, unsupervised techniques such as K-Means, Hierarchical Clustering, and DBSCAN were applied to explore relationships between budget, popularity, runtime, and vote count. Dimensionality reduction via PCA projected high-dimensional data into interpretable visual spaces, revealing natural clusters like high-budget blockbusters, critically acclaimed low-budget films, and niche mid-range productions.

Skills Demonstrated

This project demonstrates my ability to integrate supervised and unsupervised learning in a cohesive data science workflow, showcasing skills in data preprocessing, feature engineering, model training, evaluation, and visualization. It reflects my practical understanding of both classification and clustering approaches and how to structure model comparisons with consistent train-test splits.

By merging predictive modeling with exploratory analysis, the project illuminates how machine learning can not only predict financial success but also reveal hidden trends critical for decision-making in film production, marketing, and analytics.
