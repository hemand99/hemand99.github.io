Automated Reddit Comment Analysis with Transformer-Based NLP

This project focuses on building an automated system to extract and classify Reddit comments using advanced Natural Language Processing (NLP) techniques powered by transformer-based models. The goal was to analyze user discussions from any public Reddit post and classify each comment either based on sentiment (positive/negative) or content safety (NSFW vs. safe content). This system has practical applications in content moderation, sentiment analysis, and real-time social media monitoring.

To accomplish this, I used the Python Reddit API Wrapper (praw) to programmatically retrieve top-level and nested comments from a specified Reddit thread. The system cleans and parses raw comment text while filtering out unnecessary metadata. The results, along with the raw comments, were structured into a tabular format using Pandas and exported as CSV.

This end-to-end pipeline demonstrates not only predictive modeling for movie financial success by analyzing features like budget, revenue, and popularity but also leverages unsupervised learning techniques to uncover hidden patterns within movie data. It integrates advanced NLP with both supervised and unsupervised methods to deliver actionable insights for content moderation and decision-making.
