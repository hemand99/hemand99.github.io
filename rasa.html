Intelligent Chatbot using RASA Open Source & RASA X

As part of an initiative to develop real-time, intelligent conversational agents, I designed and deployed a domain-specific chatbot leveraging the RASA Open Source framework and RASA X. The project focused on delivering up-to-date COVID-19 information using natural language understanding (NLU) and dialogue management.

This chatbot achieved over 90% intent classification accuracy, enabling it to interpret and respond to user queries with high reliability. Key highlights include using domain-specific stories, custom actions with entity-based slots for context-aware replies, and iterative improvement through RASA X with real user feedback. Its deployment on a scalable VM environment makes it suitable for real-time applications in customer support, healthcare, and smart automation.

This project deepened my expertise in conversational AI, pipeline customization, and the deployment of scalable NLP solutions.
